# Diva Arcade 💅 — Which Diva Are You?

An interactive launch experience for the **Diva by Just Herbs** eau de parfum collection
— Drama Queen, Boss Babe & Desi Girl (₹469 · 50 ml · justherbs.in).

**▶ Play:** `index.html` — a 45-second alter-ego game (hold · swipe · spray) with a
shareable result card built for WhatsApp & Instagram.

**🎬 The story:** `story.html` — a cinematic scroll-driven launch film with interactive
scent pyramids for all three personas.

Fully self-contained static HTML — no build step, no dependencies. Hosted with GitHub Pages.
